Twitter Sentiment Analysis

January 21, 2021 • Posted by Varshini S

Sentiment Analysis is a natural Language processing (NLP) technique to determine whether the data is positive, negative or neutral. It is one of the widely known performed NLP tasks. This is primarily used for analyzing conversations, opinions (here, it is tweets) for deciding business strategies and assessing public opinions. There are many well known algorithms and open source packages available for prediction on sentiment analysis, however no algorithm can give 100% accuracy

HOW TO PERFORM SENTIMENT ANALYSIS ON TWITTER DATA:

  • Gather relevant twitter data
  • Clean data using pre-processing techniques
  • Create sentiment analysis ML model
  • Visualize the results

TWITTER SENTIMENT ANALYSIS IN PYTHON

There are many packages available which use different methods to do sentiment analysis. R and Python are widely used for twitter sentiment analysis.  Algorithms like SVM, Naive Bayes is used in predicting the polarity of the sentence. Open-source packages like Spacy, TextBlob, Tweepy, Vader, Flair are also available for doing sentiment analysis. We will go through some packages and methods below.

TWEEPY:

It is Python client for the official Twitter API which helps in accessing Twitter using basic authentication and OAuth. However, twitter has stopped accepting the former and only OAuth is used now.

RULE-BASED SENTIMENT ANALYSIS

Rule-based sentiment analysis is one of the very basic approaches. This method does not use any machine learning for analyzing. The sentiments of a sentence is predicted by counting the number of times the word is used in the tweet. It requires only minimal pre processing work.

TEXTBLOB:

It one of the popular Python libraries for processing textual data, stands on NLTK. Some features are:
  • Sentiment Analysis
  • Spelling Correction
  • Tokenization
  • Noun Phrase extraction
  • POS (Part Of Speech) tagging
  • N-grams,etc.

TextBlob sentiment analyzer returns two properties: 

  • Polarity is a float value that lies between [-1,1], -1 indicates negative sentiment and +1 indicates positive sentiments. 
  • Subjectivity is also a float value which lies in the range of [0,1]. Subjective sentences generally refer to personal opinion, emotion, or judgment. 

VADER:

Valence aware dictionary for sentiment reasoning (VADER) is another popular rule-based sentiment analyzer that is sensitive to both polarity (positive/ negative) and intensity (strength) of emotion. VADER sentimental analysis  relies on a dictionary that maps lexical features to emotion intensities known as sentiment scores. It returns positive, negative and neutral scores along with compound score.
